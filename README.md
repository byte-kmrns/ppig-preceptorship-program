# PPIG Preceptorship Program Website Mockup

Static website mockup for the Private Practice Interest Group's preceptorship initiative.

Live mockup: https://byte-kmrns.github.io/ppig-preceptorship-program/

Public repo: https://github.com/byte-kmrns/ppig-preceptorship-program

## Files

- `index.html` - homepage mockup
- `program.html` - program information page
- `student-signup.html` - student interest page
- `provider-signup.html` - provider interest page
- `faq.html` - full FAQ page
- `styles.css` - responsive visual system
- `app.js` - mobile navigation and static email-draft form behavior
- `assets/clinic-preceptor-hero.png` - local hero visual asset

## Notes

- The provider sign-up page preserves the current Google Form link from the existing site.
- Student/provider backup forms create email drafts because the site is static. Wire them to final Google Forms, Airtable, or another intake tool before launch.
- Copy avoids claiming guaranteed matching or official institutional approval.
- This is a static site, so it can be hosted for free on GitHub Pages from a public repository.
